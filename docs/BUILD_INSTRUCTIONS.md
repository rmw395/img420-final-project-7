# Build Instructions #
## Prerequisites ##
- Godot
- Visual Studio 2022 or newer
- Visual Studio for C++ Development

## Building ##
- Clone the repository into a personal folder.
- Once downloaded, open the folder named "module_source" in your version of Visual Studio
- Once that folder has opened, wait for the CMakeList.txt file to be fully processed by Visual Studio, indicated by the progress happening in the console
- Once that is done, build the project by either doing the hotkey `Ctrl+Shift+B` or by clicking on `Build` in the top left, and selecting `Build All`.
